# Голосовой режим - Руководство

## Возможности

Голосовой режим превращает чат в полноценный разговорный интерфейс с агентом.

### 1. Автоматическая запись после тишины
- Начните говорить в микрофон
- После 2.5 секунд тишины запись автоматически остановится
- Сообщение автоматически расшифруется и отправится агенту

### 2. Автоматическое озвучивание ответов
- Когда агент отвечает, его текст автоматически озвучивается
- После окончания озвучивания автоматически начинается новая запись
- Создает эффект непрерывного разговора

### 3. Настройки

#### Голосовой режим
Включите переключатель "Голосовой режим" для активации:
- Автоматическая отправка после паузы
- Автоматическое озвучивание ответов
- Автоматическое начало новой записи

#### Автостоп при паузе
- Определяет тишину по уровню звука < 2%
- Минимальное время записи: 1 секунда
- Время ожидания тишины: 2.5 секунды

## Использование

### Быстрый старт
1. Нажмите на иконку настроек (Activity)
2. Включите "Голосовой режим"
3. Нажмите на микрофон
4. Начните говорить
5. Сделайте паузу 2.5 секунды
6. Ответ агента озвучится автоматически

### Управление

#### Во время записи
- **Индикатор уровня звука**: показывает громкость речи
- **Красная иконка микрофона**: запись активна
- **Кнопка остановки**: остановить запись вручную

#### Во время озвучивания
- **Иконка динамика**: воспроизведение активно
- **Кнопка остановки**: прервать озвучивание
- **Автопереход**: после озвучивания начнется новая запись

### Отключение автоматики

Если нужен обычный режим:
1. Откройте настройки
2. Выключите "Голосовой режим"
3. Теперь нужно вручную нажимать кнопки записи и отправки

## Поддерживаемые форматы

### Запись
Приоритет форматов:
1. `audio/wav` - предпочтительный
2. `audio/webm` с PCM кодеком
3. `audio/mp4`
4. `audio/webm` с Opus кодеком

### Воспроизведение
- TTS API возвращает `audio/mpeg` (MP3)
- Совместимо со всеми современными браузерами

## Требования

- Микрофон с разрешением на доступ
- HTTPS соединение
- Современный браузер (Chrome, Firefox, Safari, Edge)
- Dify API ключ для STT/TTS

## Устранение проблем

### Ошибка 415 "Unsupported Media Type"
- Ваш браузер использует формат, не поддерживаемый Dify
- Попробуйте другой браузер (рекомендуем Chrome)
- Проверьте, что используется WAV формат

### Голос не распознается
- Говорите четче и громче
- Проверьте уровень звука микрофона
- Убедитесь, что микрофон работает
- Минимальное время записи: 1 секунда

### Не работает автоматика
- Проверьте, что включен "Голосовой режим"
- Проверьте, что включен "Автостоп при паузе"
- Убедитесь, что делаете паузу минимум 2.5 секунды

### Не озвучивается ответ
- Проверьте, что включен "Голосовой режим"
- Проверьте настройки звука в браузере
- Убедитесь, что Dify API ключ настроен
- Проверьте логи edge function `tts-api`

## API Integration

### STT (Speech-to-Text)
```typescript
const result = await difyClient.speechToText(audioBlob, mimeType);
// result.text содержит расшифрованный текст
```

### TTS (Text-to-Speech)
```typescript
const { data } = await supabase.functions.invoke('tts-api', {
  body: { 
    text: 'Текст для озвучивания',
    voice: 'alloy' // или другой голос
  }
});
// data.audioContent содержит base64 аудио
```

## Лучшие практики

1. **Говорите естественно**: не нужно делать искусственных пауз
2. **Четкая дикция**: агент лучше распознает четкую речь
3. **Тихая обстановка**: фоновый шум может мешать
4. **Короткие фразы**: разбивайте длинные вопросы на части
5. **Дождитесь ответа**: не прерывайте озвучивание агента
